>> Student Management System

A simple console-based Student Management System built in Python.

>> Features
- Add new students
- View all student records
- Search students by name
- Update student marks
- Delete student records
- Input validation and exception handling

>> Concepts Used
- Lists
- Dictionaries
- Tuples
- Loops
- Conditional Statements
- Exception Handling (`try-except`)
- String Methods
- User Input Handling

>> Technologies
- Python 3

>> Purpose
This project was created to practice core Python programming concepts by building a real-world CRUD (Create, Read, Update, Delete) application.
